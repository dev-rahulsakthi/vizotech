# Vizo Technologies Website

A professional, animated IT company website built with React, TypeScript, Tailwind CSS, and Framer Motion.

## Features
- **Modern UI/UX**: Clean, professional design with a focus on typography and spacing.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Animations**: Smooth scroll-reveal effects and interactive elements using Framer Motion.
- **Service Showcase**: Detailed breakdown of IT services offered.
- **Project Portfolio**: Visual grid of successful deliveries.
- **Contact Integration**: Professional contact form and company details.

## Tech Stack
- **Framework**: React 18
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Project Structure
- `src/App.tsx`: Main application component containing all sections.
- `src/index.css`: Global styles and Tailwind imports.
- `tailwind.config.js`: Custom theme and color configurations.
- `favicon.ico`: Used as the company logo throughout the site.
